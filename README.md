# 🚀 Vishu Kanoujiya - Portfolio

A modern, responsive portfolio website built with React, Tailwind CSS, and Node.js backend for contact form functionality.

## ✨ Features

- 🎨 **Modern UI/UX** with smooth animations
- 🌓 **Dark/Light Mode** toggle
- 📱 **Fully Responsive** design
- 📧 **Contact Form** with email functionality
- 🎯 **Interactive Sections** with Framer Motion
- 📊 **Skills & Projects** showcase

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS
- Framer Motion
- React Icons
- Type Animation

**Backend:**
- Node.js
- Express.js
- Nodemailer
- CORS
